# 智能衣橱搭配助手

本地 Web 应用原型，用于拍照/上传衣物、管理标签、按颜色和风格生成穿搭，并结合天气推荐当天适合的方案。

## 使用方式

打开本地服务：

```bash
python3 -m http.server 8787 --bind 127.0.0.1
```

浏览器访问：

```text
http://127.0.0.1:8787/
```

## 功能

- 衣橱入柜：支持手机拍照或批量上传图片，录入品类、主色、厚薄、风格、季节和标签。
- 照片识别：上传照片后自动分离背景，识别主色/辅助色、品类、版型、款式倾向、纹理/拼色，并生成可用于搭配的标签。
- 今日首页：打开后直接看到天气驱动的今日穿搭，可一键换一套、收藏或记录今天穿过。
- 衣橱洞察：统计品类覆盖、主色结构、穿着次数、未穿单品和最久未穿单品，帮助补齐衣橱。
- 搭配推荐：根据天气、场景、颜色协调、风格一致性、厚薄和防水标签给组合评分。
- 天气设置：支持城市天气查询、当前位置天气和手动天气。
- 收藏方案：保存喜欢的穿搭组合，后续可复穿并记录穿着次数。
- 数据导入导出：衣橱数据保存在浏览器本地，也可以导出 JSON 备份。

- `app.js`：衣橱数据、搭配规则、天气查询和交互逻辑。
